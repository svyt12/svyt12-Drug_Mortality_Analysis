# svyt12-Drug_Mortality_Analysis

Taylor's University Data Mining Assignment

The “Accidental Drug Related Deaths” dataset provides an overview of the opioid crisis. The objective is to transcend basic descriptive statistics by converting this raw mortality data into actionable public health insights by applying data mining techniques such as regression, clustering, anomaly detection, and classification.
Additionally, the significance of this research lies in its multi-dimensional approach to understanding drug-related mortality. Ultimately, these findings serve as a statistical foundation for resource allocation and harm-reduction programs tailored to the unique, non-linear dynamics of the current drug epidemic.
